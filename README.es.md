# Frontend Mentor - News Homepage Solution

Esta es mi solución al desafío **News Homepage** de Frontend Mentor. Este proyecto se centra en la construcción de una página principal de noticias responsiva, con un diseño editorial limpio, navegación interactiva y buenas prácticas modernas de frontend utilizando HTML, CSS y JavaScript.

El reto fue una excelente oportunidad para practicar técnicas de maquetación con CSS Grid y Flexbox, diseño responsive e interactividad básica de la interfaz de usuario.

## Visión general
Este proyecto es una página principal de noticias responsiva que se adapta de forma fluida a dispositivos de escritorio, tablet y móviles. Presenta un diseño editorial de varias columnas en pantallas grandes, un diseño de una sola columna optimizado para móviles y un menú hamburguesa interactivo para dispositivos pequeños.

El objetivo fue replicar el diseño proporcionado lo más fielmente posible, manteniendo el código limpio, semántico y fácil de mantener, siguiendo estándares modernos de desarrollo frontend.

---

## El desafío
Los usuarios deben poder:

- Ver el diseño óptimo según el tamaño de la pantalla de su dispositivo.
- Visualizar diferentes disposiciones para escritorio y móvil mediante técnicas de diseño responsivo.
- Abrir y cerrar el menú de navegación móvil usando un ícono de menú hamburguesa.
- Experimentar transiciones suaves en los estados de la interfaz, como overlays y efectos hover.
- Navegar por una interfaz que se aproxime fielmente al diseño original de Frontend Mentor.

## Mi proceso
- Estructuré la página utilizando **elementos semánticos de HTML5** como `header`, `nav`, `main`, `section`, `article` y `aside`.
- Diseñé la maquetación principal usando **CSS Grid** para la estructura editorial y **Flexbox** para la alineación de componentes.
- Implementé **diseño responsivo** mediante media queries para adaptar el layout a distintos puntos de ruptura.
- Utilicé **propiedades personalizadas de CSS (variables)** para gestionar colores y mantener consistencia visual.
- Apliqué la **metodología BEM** para el nombrado de clases, facilitando la escalabilidad y el mantenimiento del CSS.
- Construí un menú de navegación móvil interactivo usando **JavaScript puro**, controlando los estados de la interfaz mediante clases CSS.
- Implementé un overlay de fondo utilizando pseudo-elementos de CSS para mejorar la experiencia de usuario del menú móvil.
- Aseguré estados hover y focus para mejorar la usabilidad y la accesibilidad.

## Lo que aprendí
- Cómo construir layouts editoriales complejos utilizando **CSS Grid** manteniendo la responsividad.
- Combinar **Grid y Flexbox** de forma efectiva dentro de un mismo proyecto.
- Gestionar estados de la interfaz (abrir/cerrar menú, overlays) usando JavaScript y clases CSS.
- Crear patrones de navegación responsiva.
- Mejorar la mantenibilidad del código mediante **variables CSS** y convenciones **BEM**.
- Potenciar la experiencia de usuario a través de interacciones sutiles como overlays, efectos hover y transiciones.
- Estructurar proyectos frontend de forma limpia y escalable, alineados con escenarios del mundo real.
